Curwe Chat Manager - Документация

Добро пожаловать в документацию Curwe Chat Manager — многофункционального Telegram бота для управления чатами с поддержкой русского и английского языков, системой уровней доступа и широкими возможностями модерации.

Основные возможности

• Управление пользователями (варны, муты, баны)
• Система уровней доступа (от 0 до 5)
• Поддержка русских и английских команд
• Статистика чата и пользователей
• Настраиваемые приветственные сообщения
• Функция блокировки чата
• Кастомные алиасы команд

Команды бота

Бот поддерживает команды на русском и английском языках, а также алиасы (синонимы) для удобства использования. Команды можно вызывать через "/" (например, /warn) или через "+" для русских команд (например, +варн).

Система уровней доступа

В Curwe Chat Manager реализована система уровней доступа от 0 до 5. Каждый уровень определяет, какие команды может использовать пользователь.

0
Пользователь

Обычные участники чата. Могут использовать базовые команды: menu, profile, help, report.

1
Помощник

Могут просматривать списки варнов, мутов, банов. Могут закреплять сообщения.

2
Модератор

Могут выдавать варны, муты, кикать пользователей, очищать сообщения, блокировать чат.

3
Администратор

Могут банить пользователей, размучивать, настраивать чат, устанавливать описания другим.

4
Старший админ

Могут разбанивать, повышать/понижать уровни других пользователей.

5
Владелец

Полный доступ ко всем функциям, включая отладку, перезапуск, управление алиасами.

Как назначаются уровни?

Уровень 0 присваивается автоматически всем новым пользователям.
Уровень 1-2 могут быть назначены администраторами через команды /promote и /demote.
Уровень 3 автоматически присваивается администраторам чата в Telegram.
Уровень 5 только для владельца бота (указывается в config.json).

Настройка бота

Для начала работы с ботом выполните следующие шаги:

Быстрая настройка

1. Добавьте бота в чат и дайте ему права администратора
2. Автоматическая инициализация — бот создаст базу данных
3. Настройте приветствия командой /welcomeset
4. Назначьте уровни модераторам через /promote
5. Настройте параметры в /settings

Важные команды для настройки

Команда Уровень Описание
/dbcreate 5 Создать базу данных для чата (если автоматически не создалась)
/welcomeset 5 Настроить приветственное сообщение для новых участников
/settings 3 Настройки чата (авто-варны, авто-муты, приветствия)
/promote @user 4 Повысить уровень пользователя
/demote @user 4 Понизить уровень пользователя

Использование команд

Бот поддерживает несколько форматов команд для удобства использования:

Английский формат
/warn @username /mute @username 7d /ban @username спам

Классический формат через слеш

Русский формат (+)
+варн @username +мут @username 7d +бан @username спам

Русские команды через плюс

Русский формат (-)
-варн @username -мут @username -бан @username

Отмена действий через минус

Примеры использования

Выдать предупреждение:
/warn @username нарушение правил или +варн @username нарушение правил

Замутить на 3 дня:
/mute @username 3d или +мут @username 3d

Забанить за спам:
/ban @username спам или +бан @username спам

Проверить блокировку чата:
/chatstatus или /статусчата

Часто задаваемые вопросы

Как добавить бота в чат?

1. Найдите бота @curwechatmanagerbot в Telegram
2. Нажмите "Добавить в группу"
3. Выберите чат и дайте боту права администратора
4. Бот автоматически настроится и отправит приветственное сообщение

Как работает блокировка чата?

При блокировке чата командой /lockchat обычные пользователи (уровень 0) не могут отправлять сообщения. Модераторы и администраторы могут писать как обычно. Это полезно для временной остановки обсуждений.

Что такое кастомные алиасы?

Алиасы — это пользовательские сокращения для команд. Например, вы можете создать алиас +штраф для команды /warn. Управление алиасами доступно через команду /alias (требуется уровень 5).

Как сбросить настройки чата?

Используйте команду /dbcreate для пересоздания базы данных. Внимание: это удалит всю статистику и настройки чата.
